# Case Study Generator

Single-file, no-build HTML generator for anonymized performance marketing case studies.

## Files

- `index.html` — GitHub Pages entry point
- `case-study-generator.html` — source filename
- `client-case-study-generator.html` — legacy alias
- `CODEX_INSTRUCTIONS.md` — design principles and constraints for future edits

## Core Rules

- Blank inputs mean "not tracked," never zero.
- No client names or client logo fields.
- Outcome callout is the safe default; testimonials require client approval.
- Engagement focus controls fields, metrics, copy, hero stats, and tables.
- Output remains a one-page A4 PDF via print/export.
